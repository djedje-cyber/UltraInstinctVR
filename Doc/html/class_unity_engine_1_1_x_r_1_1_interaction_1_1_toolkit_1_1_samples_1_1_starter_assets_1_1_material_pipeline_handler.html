Detailed Description

Scriptable object that allows for setting the shader on a material based on the current render pipeline. Will run automatically OnEnable in the editor to set the shaders on project bootup. Can be refreshed manually with editor button. This exists because while objects render correctly using shadergraph shaders, others do not and using the standard shader resolves various rendering issues.
