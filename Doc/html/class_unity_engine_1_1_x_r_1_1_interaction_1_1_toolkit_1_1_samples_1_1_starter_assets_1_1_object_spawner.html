◆ TrySpawnObject()

bool UnityEngine.XR.Interaction.Toolkit.Samples.StarterAssets.ObjectSpawner.TrySpawnObject ( Vector3 spawnPoint,
Vector3 spawnNormal )

Attempts to spawn an object from objectPrefabs at the given position. The object will have a yaw rotation that faces cameraToFace, plus or minus a random angle within spawnAngleRange.

Parameters
spawnPointThe world space position at which to spawn the object.
spawnNormalThe world space normal of the spawn surface.
Returns
Returns true if the spawner successfully spawned an object. Otherwise returns false, for instance if the spawn point is out of view of the camera.

The object selected to spawn is based on spawnOptionIndex. If the index is outside the range of objectPrefabs, this method will select a random prefab from the list to spawn. Otherwise, it will spawn the prefab at the index.
